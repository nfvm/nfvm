1. 安装方式

1.1. 下载 whl 文件后安装

在whl文件所在目录下通过下面的命令安装

pip install nfvm-0.1.0-py3-none-any.whl

卸载可以通过命令

pip uninstall nfvm

1.2. 以开发者模式使用

以开发者模式使用，相当于将源码文件夹添加到python的库搜索路径中，优点是修改源码后无需再次安装。 安装命令为

pip setup.py develop

卸载命令为

# 可能因为 setuptools 模块版本问题而失败，手动删除文件夹即可
pip setup.py develop --uninstall

1.3. 打包为 whl

如有必要，可以通过下述命令打包为 whl 文件

# 安装 wheel 模块用于打包
pip install wheel

# --no-deps: 不下载依赖
# -w dist: 指定生成文件路径
pip wheel --no-deps  -w dist .
